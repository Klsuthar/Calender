Cloud Database Integration Guide

By default, AuraCal saves all data instantly to your browser's local storage (LocalStorage). This means data is 100% private and offline.

Want to sync across devices or databases?

You can connect Firebase, Google Sheets, or Supabase. A complete integration guide has been generated for you in the repository:

database_guide.md

Quick Firebase Setup Example:

// Initialize Firebase in app.js
import { initializeApp } from "https://www.gstatic.com/firebasejs/10.8.0/firebase-app.js";
import { getDatabase, ref, set, onValue } from "https://www.gstatic.com/firebasejs/10.8.0/firebase-database.js";

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  databaseURL: "https://YOUR_PROJECT-default-rtdb.firebaseio.com",
  projectId: "YOUR_PROJECT"
};
const app = initializeApp(firebaseConfig);
const db = getDatabase(app);

Please check the file database_guide.md in this workspace for a complete tutorial, code templates, and step-by-step instructions on connecting to Google Sheets or Firebase Realtime Database!
